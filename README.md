# Space-Platformer-Game
Space Odyssey – The Afterlife of a Crewmate (2D Game) — Java:
- Developed a 2D endless platformer game in Java with GUI, featuring player movement, collision detection, gravity, camera
tracking, difficulty scaling, and a binary-based scoring system
- Implemented object-oriented programming with inheritance and encapsulation.
- Incorporated sound effects and animated sprites for power-ups, monsters, and game actions.
- Built interactive features such as a settings panel, pause function, event-driven controls, and persistent high score saving for
an engaging gameplay experience.
